# 🌐 Multilingual Chatbot Frontend (React)

A simple React app that connects to a multilingual chatbot backend.

## 🧰 Setup
```bash
npm install
```

## ⚙️ Environment
Create `.env` with:
```
REACT_APP_API_URL=https://your-backend-url.onrender.com/chat/
```

## 🚀 Run Locally
```bash
npm start
```

## 🛰️ Deploy to Vercel
Import this repo → Add env var → Deploy!
